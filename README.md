# un7z

一个用于批量递归解压的 Bash 脚本，支持选择性解压并提供单行进度条显示，避免滚屏。

## 特性

- 递归扫描子文件夹中的压缩包
- 支持多种格式：7z/zip 分卷、rar 分卷、tar.gz/tgz
- 解压前完整列出归档，支持选择编号或范围
- 单行进度条显示（不滚屏）
- CRC/测试失败自动记录并跳过

## 依赖

- `bash`（macOS 自带 3.2 也可运行）
- `find`, `tar`
- `7zz`（7-Zip）
- `unrar`

> macOS 可用 Homebrew 安装：`brew install p7zip unrar`

## 使用方法

在脚本所在目录执行：

```bash
./un7z
```

流程：
1. 递归扫描压缩包并列出
2. 选择要解压的编号（如 `1,3,5-7`），或直接回车使用 `all`
3. 输入密码（没有则回车）
4. 显示单行进度条，逐个解压

## 支持的文件类型

- `*.7z.001`
- `*.zip.001`
- `*.part*.rar`（仅处理 `part01` 起始卷）
- `*.tar.gz`
- `*.tgz`

## 输出与日志

- `extract.log`：解压/测试日志
- `failed.log`：失败归档列表

## 退出状态

- 无归档时退出码为 0
- 其它错误按脚本输出提示处理
